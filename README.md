# 西瓜忍者｜60秒挑战

一个可交给 Codex 继续开发的切水果手机网页小游戏骨架。

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- localStorage
- Vercel

## 快速开始

```bash
npm install
npm run dev
```

打开：

```text
http://localhost:3000
```

## 验证

```bash
npm run lint
npm run build
```

## 核心玩法

- 60 秒倒计时
- 滑动切水果
- 水果加分
- 炸弹扣分
- Combo 加分
- 最高分保存到 localStorage

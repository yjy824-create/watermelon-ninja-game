# 交给 Codex 的第一段指令

请先阅读本项目的 `AGENTS.md`、`README.md`、`docs/game-design.md`、`docs/deployment-checklist.md`、`prompts/image2-assets.md`。

你的任务：
1. 安装依赖并运行项目。
2. 修正所有 TypeScript / ESLint / Build 错误。
3. 确认游戏 MVP 可以运行。
4. 优化手机端操作体验。
5. 保持当前游戏主题：西瓜忍者｜60秒挑战。
6. 不要删除 AGENTS.md、docs、prompts。
7. 完成后执行 `npm run lint` 和 `npm run build`。
8. 如果用户提供 GitHub 仓库，请 push 到 main。
9. 如果用户要求部署，请协助部署到 Vercel。

完成后按 AGENTS.md 的回报格式说明结果。
